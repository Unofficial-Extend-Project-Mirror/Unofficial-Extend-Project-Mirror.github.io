Welcome to the Unofficial-Extend-Project-Mirror project

About this project

This project unofficially mirrors the source code repositories considered most important from Extend Project's Git and SVN repositories.

This has been created as an independent effort, in an attempt to avoid future situations as the one that occurred for the first part of 2013: thread "OF 1.6-ext Repository offline?", post #2 at CFD-Online.com/Forums

How to recreate this mirror project

For those who wish to replicate the mirrors created here, this is the outline of the steps taken so far:

  1. On a working Linux Distribution, the commands used are provided here: Extend-clonings.txt

  2. Then the following script was used for hooking up the local clones of the repositories to the repositories (manually) created at Github: associateReposToGithub.sh

For updating the local and remote mirrors later on:

  1. For updating the local mirrors: pushAllRepos.sh

  2. For then uploading the the mirrors at Github: pullAllRepos.sh
