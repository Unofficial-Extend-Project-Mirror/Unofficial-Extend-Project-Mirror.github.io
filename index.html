Welcome to the Unofficial-Extend-Project-Mirror project

This project unofficially mirrors the source code repositories considered most important from Extend Project's Git and SVN repositories.

This has been created as an independent effort, in an attempt to avoid future situations as the one that occurred for the first part of 2013: http://www.cfd-online.com/Forums/openfoam-installation/111630-1-6-ext-repository-offline.html#post401396

All of the source code provided in these mirrored repositories is GPL (GNU Public License), although versions may differ depending on the mirrored source code folder/repository.
