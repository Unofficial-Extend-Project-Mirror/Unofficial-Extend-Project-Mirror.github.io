Welcome to the Unofficial-Extend-Project-Mirror project

This project unofficially mirrors the source code repositories considered most important from Extend Project's Git and SVN repositories.

This has been created as an independent effort, in an attempt to avoid future situations as the one that occurred for the first part of 2013: thread "OF 1.6-ext Repository offline?", post #2 at CFD-Online.com/Forums

The license for all of the source code provided in these mirrored repositories is GPL (GNU Public License), although versions may differ depending on the mirrored source code folder/repository. Please refer to the README* files and source code files therein, as well as the original repositories, for more information.
